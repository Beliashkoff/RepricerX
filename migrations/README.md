# Миграции

Накатываются `golang-migrate/migrate` (см. `cmd/migrate`). Версия — целое число
из имени файла (`NNNNNN_*.up.sql`).

## Соглашения

- Имя: `NNNNNN_<scope>.up.sql` + парный `*.down.sql`.
- Идемпотентность: `IF NOT EXISTS` / `IF EXISTS`, `DO $$ ... EXCEPTION WHEN duplicate_object`.
- Время: `TIMESTAMPTZ`, дефолт `NOW()`.
- UUID: `gen_random_uuid()` (требует `pgcrypto`, подключается в 000001).
- Большие/долгие операции (бэкфилы, переиндексация) — отдельной миграцией.

## Заметки по нумерации

- Версия **000007** пропущена намеренно: она использовалась во время
  рефакторинга этапов 5–6 и была отменена до релиза. Перенумерация
  `000008…000013` под 000007 не делается, потому что миграции уже накатаны
  на VPS и в `schema_migrations` записаны соответствующие версии — попытка
  понизить номер привела бы к неконтролируемому ролбэку.
